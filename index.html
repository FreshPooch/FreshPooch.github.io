My Projects

Pizza Amore

Website

GitHub

Built with JavaScript, HTML, CSS, Node.js, Express, and Heroku. This website is a pseduo ecommerce site where a user can create a custom order from the menu items and save it to a personal cart. The user is then able to change or delete specific items in their cart before placing an order. The site also allows a user to place a catering request. The site is styled to reflect an old-fashioned Italian restaurant. Mobile device users will find the functionality of the website to be completely responsive.
